# Valley Catholic Basketball Data Upload

This guide walks you through uploading all the Valley Catholic basketball team data to your database.

## Prerequisites

1. **Supabase Tables Must Exist** - First, ensure you've run the latest SUPABASE_SETUP.sql in your Supabase SQL Editor
2. **Environment Variables** - Your `server/.env` file must have valid Supabase credentials:
   ```
   SUPABASE_URL=https://xxx.supabase.co
   SUPABASE_ANON_KEY=your_anon_key
   ```

## Verify Data Upload

After running the seed script:

1. Go to [Supabase Dashboard](https://supabase.com)
2. Select your project
3. Click **Table Editor** (left sidebar)
4. Check these tables have data:
   - `teams` - Should have 2 rows (Valley Catholic Boys, Valley Catholic Girls)
   - `players` - Should have 25 rows total
   - `games` - Should have 50 rows total

## View in App

After seeding:

1. Go to your app at `https://your-domain.com`
2. Click **Teams** button in navbar
3. You should see:
   - Valley Catholic Boys Basketball (Rank 3)
   - Valley Catholic Girls Basketball (Rank 8)
4. Click each team to see full roster with player details and bios

## Notes

- All schedules are set to auto-populate from Supabase
- Player bios are immutable by users (admin-only editing capability planned)
- Game results can be updated via admin panel once feature is built
- All heights are stored as strings for flexibility
- Timestamps (created_at, updated_at) are auto-generated by Supabase
